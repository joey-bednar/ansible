# Install

Run `./install [option]` to setup a new computer. Tested on `Ubuntu 22.04.04 (Jammy)` and `Ubuntu 24.04.04 (Noble)`.

# Test

Run `./install container` to build and run a Docker image with `nvim`, `zsh`, and `tmux`.
